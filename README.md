# jquery-kindeditor-suggest
在富文本输入区域（kindeditor）可以有@功能，能够容易提示用户名的。在网上找了一个叫bootstrap-suggest，却不能满足我的需求，于是我决定在该插件上加上改良，主要是下面几点：

1. @内容的输入，能够匹配多个属性值。
2. 选中列表值后，是成块插入kindeditor的（方便删除，也利于区分文本）。
3. 修正偶数行@功能失效。
